How to complete a task?

  1. Find your favourite task from the Tasks List in this site.

  2. For more details click on more details section which is shown below in every task. After reading the description (more details) about the task and understanding it you can start working on that.
  3. Follow the instructions in .pdf file which relates to the task. Complete your work as given in the instructions.
  4. Save your work as given in the description of the task.

  5. Send us a link of your work using the form that is shown bottom of this page or email directly to bits.acicts@gmail.com as per the instructions given in the task document.
  6. We'll send you a reply email informing you whether we accept it or not. Until then, you can start working on a new task.
  7. If your task got accepted, you can start your work on completing another task. If not you can correct your mistake (We'll notice you about the mistake) and then you can submit your work again or you can select  another task to complete.
